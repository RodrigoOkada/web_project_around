# Tripleten web_project_around

## Visão Geral do Projeto

Este projeto é um mockup de uma interface de uma rede/galeria de fotos, focada em exibir um perfil de usuário e uma coleção de cartões de imagem interativos. O projeto simula a funcionalidade de edição de perfil através de um modal (popup) e exibe uma lista estática de cartões de fotos.

## Tecnologias Utilizadas

- HTML5: Estrutura semântica da página.

- CSS3: Estilização da interface. O projeto segue a metodologia BEM (Block, Element, Modifier) para a nomenclatura de classes, garantindo modularidade e escalabilidade.

- JavaScript (Vanilla JS): Implementação da lógica de interatividade, especificamente para a manipulação do modal de edição de perfil.
